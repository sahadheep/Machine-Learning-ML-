# Titanic Data Preprocessing with Pandas

## 📌 Objective
Learn the fundamentals of data preprocessing using Pandas by cleaning and analyzing the Titanic dataset.

## 📂 Dataset
- Titanic Dataset (CSV)

## 🛠 Topics Covered
- Loading CSV files
- Inspecting the dataset
- Handling missing values
- Checking and removing duplicates
- Filtering rows
- Creating new columns (Feature Engineering)
- Grouping data using `groupby()`
- Merging DataFrames
- Calculating descriptive statistics
- Saving the cleaned dataset

## 🚀 Feature Engineering
Created the following new features:
- `FamilySize`
- `IsAlone`
- `AgeGroup`

## 📊 Skills Practiced
- Data Cleaning
- Boolean Indexing
- Data Filtering
- Data Aggregation
- Feature Engineering
- Dataset Export

## 📁 Output
- `cleanedfile.csv`

## 📚 Libraries Used
- Pandas
- numpy
